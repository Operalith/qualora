# Qualora v0.3.0-alpha

Qualora v0.3.0-alpha adds a minimal self-hosted web UI and human-friendly HTML reports on top of the browser/API worker foundation.

## Highlights

- New `qualora-web` service on `http://localhost:3000`.
- Project list, project creation, and project detail views.
- Run list and run report views.
- Findings, evidence metadata, browser metadata, API metadata, and worker job metadata in the UI.
- Self-contained HTML report export at `GET /api/v1/runs/{run_id}/report.html`.
- Run listing endpoints for all runs and project-scoped runs.
- CI and Makefile coverage for the web build/type-check.
- Smoke script now prints JSON and HTML report URLs.

## Safety And Scope

- No authentication or authorization yet.
- Intended for trusted local/self-hosted alpha environments only.
- API checks remain safe by default.
- Destructive API testing remains unsupported.
- Active security scanning is not included.

## Known Limitations

- UI is alpha and intentionally minimal.
- No evidence object proxy or screenshot preview/download through the API yet.
- No project editing, pagination, or advanced report filtering.
- No authenticated API testing or login automation.
- No multi-tenancy, teams, billing, or SaaS assumptions.
